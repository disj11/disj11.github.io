# Agent Guidelines for disj11.github.io

## 프로젝트 개요

Hugo 기반의 기술 블로그입니다. 블로그 포스트는 `content/posts/` 디렉토리에 마크다운 파일로 작성됩니다.

## 블로그 포스트 작성 규칙

### 프론트매터 (Frontmatter)

모든 블로그 포스트 파일(`content/posts/*.md`)은 반드시 YAML 프론트매터를 포함해야 합니다.

**필수 필드:**

```yaml
---
title: "포스트 제목"
description: "포스트 내용을 요약하는 설명"
date: YYYY-MM-DDTHH:MM:SS+09:00
tags: ["tag1", "tag2"]
---
```

- `title`: 포스트의 제목. 큰따옴표로 감쌉니다.
- `description`: 포스트의 내용을 한두 문장으로 요약. 큰따옴표로 감쌉니다.
- `date`: ISO 8601 형식의 날짜. 타임존은 `+09:00` (KST)을 사용합니다.
- `tags`: 관련 태그 목록. 소문자 영문으로 작성하며, 하이픈(`-`)으로 단어를 구분합니다.

**선택 필드:**

- `lastmod`: 마지막 수정일. `date`와 동일한 형식을 사용합니다.
- `url`: 커스텀 URL 경로. 기본값은 파일명 기반이므로 특별한 경우에만 사용합니다.

### 콘텐츠 언어

- 포스트는 한국어로 작성합니다.
- 기술 용어는 원문(영문)을 그대로 사용하거나 한글 표기 후 괄호 안에 영문을 병기합니다.
